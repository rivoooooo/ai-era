## Overview

This guide lists all keyboard shortcuts available in AI-Era to help you navigate more efficiently.

## Global Shortcuts

These shortcuts work across the entire platform:

- `/` - Open search
- `Cmd+K` - Open search (Mac)
- `Ctrl+K` - Open search (Windows/Linux)
- `?` - Show keyboard shortcuts
- `Esc` - Close dialogs

## Docs Page Shortcuts

When viewing documentation:

- `[` - Previous document
- `]` - Next document
- `g g` - Scroll to top
- `G` - Scroll to bottom
- `↑` `↓` - Navigate in tree/search

## Challenge Shortcuts

During challenges:

- `Cmd+S` - Save code
- `Cmd+Enter` - Run code
- `Ctrl+S` - Save code (Windows/Linux)
- `Ctrl+Enter` - Run code (Windows/Linux)
